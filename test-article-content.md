# Test Artikel dengan Markdown

## Pendahuluan

Ini adalah **contoh artikel** dengan berbagai *formatting markdown* untuk menguji komponen ArticleContent yang baru.

### Fitur yang Diuji

1. **Bold text** dan *italic text*
2. Lists (ordered dan unordered)
3. Links dan headings
4. Blockquotes dan code blocks

## Section Kedua

Berikut adalah contoh list:

- Item pertama
- Item kedua dengan **bold**
- Item ketiga dengan *italic*

### Ordered List

1. Langkah pertama
2. Langkah kedua
3. Langkah ketiga

## Blockquote

> Ini adalah contoh blockquote yang akan di-render dengan styling khusus.
> Blockquote bisa multi-line seperti ini.

## Code Block

Contoh inline code: `const example = "test"`

## Link

Kunjungi [Teras Dakwah](https://example.com) untuk informasi lebih lanjut.

## Kesimpulan

Artikel ini menguji semua fitur markdown yang didukung oleh komponen baru.
